# COVID and Bacterial Pneumonia Detector

In this repo, we train a deep learning model to distinguish between x-ray images of healthy individuals and those with COVID-19 or bacterial pneumonia.


# Test it out

To test out this model, please visit our small user-friendly Github repo at [https://github.com/vivianngo97/COVID-Bacterial-Pneumonia-Classifier-Run-Model](https://github.com/vivianngo97/COVID-Bacterial-Pneumonia-Classifier-Run-Model).
